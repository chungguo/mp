# 健牙宝 - 口腔健康小程序

一个基于 MPX 框架开发的口腔健康服务小程序，提供口腔咨询、AI牙齿检测、医院推荐、优惠券等功能。

## 📱 项目概述

健牙宝是一款专注于口腔健康服务的小程序，为用户提供：
- 🏥 **合作医院推荐** - 基于地理位置的优质口腔医院推荐
- 🤖 **AI牙卫士** - 智能牙齿健康检测与分析
- 🦷 **口腔咨询** - 专业口腔问题咨询服务
- 🎫 **优惠活动** - 特惠洗牙券和新用户福利
- 👥 **推荐活动** - 用户推荐奖励机制

## 🛠 技术栈

### 核心框架
- **MPX 2.10.17** - 小程序开发框架
- **Vue 2.7.16** - 前端框架
- **TypeScript 4.9.5** - 类型支持

### 状态管理
- **Pinia 2.3.1** - 现代化状态管理
- **@mpxjs/pinia** - MPX Pinia 集成

### UI 组件库
- **TDesign Miniprogram 1.11.1** - 腾讯 TDesign 小程序组件库

### 样式方案
- **UnoCSS** - 原子化 CSS 框架
- **Less** - CSS 预处理器
- **PostCSS** - CSS 后处理

### 工具链
- **ESLint** - 代码质量检查
- **Babel** - JavaScript 编译
- **Webpack 5.102.1** - 模块打包

## 🚀 快速开始

### 环境要求
- Node.js >= 14
- pnpm (推荐) 或 npm

### 安装依赖
```bash
pnpm install
```

### 开发调试
```bash
# 启动小程序开发模式
pnpm run serve
```

### 生产构建
```bash
# 构建生产版本
pnpm run build
```

### 代码检查
```bash
# 运行 ESLint 检查
pnpm run lint
```
